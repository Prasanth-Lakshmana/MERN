Html tags

  • DOCTYPE - version of html
  • head - tab title and meta
  • body - include contents of the page
  • p - paragraphs
  • ul - unordered lists, ol - ordered lists
  • img - loads a image using src attribute
  • table - creates a table
    • thead,tbody - optional
    • th - create a column heading
    • tr - create a new row
    • td - insert data in row
  • a - anchor tag used to link pages
